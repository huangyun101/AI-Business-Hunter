# 程序员副业小生意全景手册

一个会写代码的人，怎么从零开始做互联网小生意。这本手册从心理、策略、技术、信息源四个维度，按入门/中阶/高阶三档规模，把整条路讲清楚。

手册里的案例与方法，全部提炼自我加入的各类社群中的真实复盘，含生财有术社群、AI 破局、出海圈实战记录、YouTube AI 工具生态一线经验。案例中的收入数字均为分享者自述，只用来说明路径量级，不构成收益承诺。

## 目录

- **第一章 全景图：程序员能做的互联网小生意地图**：六条主干道，三档规模，怎么选第一条路
- **第二章 心理建设：开工之前先修的内功**：不敢开始、坚持不住、身份转变，三关的破法
- **第三章 需求发现与验证：生意从哪里来**：需求四来源、五问框架、30 分钟验证、MVP 边界
- **第四章 入门级变现（月入几百~几千）**：接单、新生态服务、虚拟产品，赚到验证的钱
- **第五章 中阶变现（月入几千~几万）**：工具站、插件、小程序、AI 产品，开始拥有资产
- **第六章 高阶变现（月入几万以上）**：SaaS、产品矩阵、B 端大单，靠系统赚钱
- **第七章 技术层面：每个阶段该会什么、不该学什么**：建站流水线、支付接入、模型成本工程
- **第八章 流量与获客：SEO、平台、社群冷启动**：三种流量的性格与打法
- **第九章 定价与收款：从不敢收钱到定价自信**：C 端档位、B 端算账、订阅定价、收款通道
- **第十章 信息源与情报系统：持续找到下一个机会**：信息源清单、情报流水线、社群用法
- **第十一章 杠杆与放大：自动化、外包、个人 IP**：三根杠杆的用法和上杠杆的顺序
- **第十二章 避坑清单与长期主义**：方向、合规、执行三类坑，四个复利账户

## 推荐《生财有术》

生财有术有着一群爱拼爱整活的圈友，他们在各个平台上都努力着赚钱。小红书，抖音，B站，知乎，微信生态圈，Reddit, X，TikTok, 等等。在这个圈子里，能看到所有不上班却还能养活自己的机会。本手册案例绝大部分都来自这里，推荐你也加入。
